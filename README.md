# BOOKSHOP

`BOOKSHOP` is a bookshop client for fetching and adding bookshop.

## To Run Locally

### Clone repository

```bash
git clone https://github.com/braswelljr/bookshop.git
```

### Install dependencies

```bash
yarn install
# or
npm install
```

### Run command

```bash
yarn run dev
# or
npm run dev
```
